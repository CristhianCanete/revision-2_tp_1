  Las fuentes son un elemento indispensable para cualquier diseño y por ello, los profesionales están siempre a la caza de las tipografías que mejor se adapten a sus proyectos.
  Fonts Ninja es una herramienta de diseño que permitirá a los diseñadores gráficos y desarrolladores web probar un número de tipografías suficiente, compatible con cualquier navegador que soporte las extensiones de chrome, antes de obtener (comprar) finalmente la tipografía adecuada para cada uno de sus trabajos en desarrollo.


Utilizacion de la extensión






  El beneficio de instalar esta extensión es que cada vez que veamos algun tipo de letra atractivo en una web, la capturemos y la añadamos a nuestro   Fonts Ninja para que este disponible poder probarlo junto con el resto.
Muchas veces navegamos entre fuentes y creemos que una es la ideal para nuestro proyecto, pero una vez la usamos, no la encontramos tan atractiva como cuando la vimos en la web. Fonts Ninja nos permite
